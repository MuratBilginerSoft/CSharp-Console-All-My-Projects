## Kapsanan Konular

- **Temel Kavramlar:** Değişkenler, veri tipleri, tür dönüşümleri, operatörler
- **Kontrol Yapıları:** If-Else, Switch-Case, ternary operatör
- **Döngüler:** For, While, Do-While, foreach
- **Diziler:** Tek boyutlu diziler, çok boyutlu diziler, matris işlemleri
- **Koleksiyonlar:** ArrayList, Hashtable
- **Metotlar:** Fonksiyon tanımlama, parametre kullanımı, geri dönüş değerleri
- **OOP:** Sınıflar (Class), yapılar (Struct), nesne oluşturma
- **Algoritmalar:** Fibonacci, Tribonacci, Pascal üçgeni, faktöriyel, permütasyon, kombinasyon, palindrom, asal sayı bulma
- **Uygulamalar:** Hesap makinesi, lokanta otomasyonu, sayı tahmin oyunu, şifreleme, loto

## Gereksinimler

- Visual Studio 2010 veya üzeri
- .NET Framework 4.0 veya üzeri

## Kullanım

Her bir proje klasöründeki `.sln` dosyasını Visual Studio ile açarak projeyi çalıştırabilirsiniz.
